=== Mason WordPress: "Page Links To" Plugin Sitemap Fix ===
Contributors: Jan Macario
Requires at least: 4.7
Tested up to: 6.1
Requires PHP: 7.0
Stable tag: 0.9

Mason WordPress plugin which fixes the sitemap for sites using the Page Links to WordPress plugin. Specifically, it excludes posts which are set to link to another URL using the Page Link To plugin.
